Energy Gauntlet API

The Energy gauntlet data can be accessed via either http and web sockets.

Both http and web sockets can access the raw data from the energy gloves and the commands for the Double Telepresence Robot.

HTTP

Data from the Energy Gauntlets can be retrieved at the urls:

  • Raw Data
  • Command Data

Web Sockets

Data from the Energy Gauntlets can also be streamed over web sockets at:

  • ws:///ws/raw
  • ws:///ws/commands

Raw Data

The raw data API returns the raw and unprocessed data directly from the sensors on the gloves.
